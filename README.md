# 🚀 TrendPilot: K-Beauty 글로벌 진출 전략 & 마케팅 에이전트

**TrendPilot**은 K-뷰티 브랜드가 글로벌 시장에 진출할 때 직면하는 현지 규제, 성분 리스크, 시장 트렌드 파악의 어려움을 AI로 해결해주는 B2B 전문 에이전트 파이프라인입니다.
제품 사진 1장과 목표 국가만 입력하면, **비전 모델(Vision)**이 제품 속성을 파악하고, **전문 검색(Local RAG)**과 **트렌드 검색(Web RAG)**을 거쳐 **시장 진입 전략 보고서 및 현지화 맞춤 광고 에셋**을 한 번에 자동 생성합니다.

---

## 🌟 파이프라인 아키텍처 (5-Step Agent)

1. **Vision Agent (`vision_agent.py`)**
   - 제품 이미지를 분석하여 화장품 카테고리, 타깃 층, 용기 재질 및 무드(Tone & Manner)를 도출합니다.
2. **Query Router (`rag_router.py`)**
   - Vision 결과를 기반으로, 로컬 DB(규제/통계)와 웹(경쟁사/트렌드)을 탐색하기 위한 검색 쿼리를 동적으로 생성합니다.
3. **Dual RAG Agent**
   - **Local RAG (`local_rag_agent.py`)**: 회사 내부에 구축된 뷰티 수출 통계 및 국가별 화장품 성분 규제/가이드라인(PDF/CSV)을 검색하여 리스크를 분석합니다.
   - **Web RAG (`web_rag_agent.py`)**: 구글 트렌드(Google Trends)와 네이버 검색 크롤링을 활용하여 현지 경쟁사 동향 및 최신 밈/키워드를 수집합니다.
4. **Strategy Agent (`strategy_agent.py`)**
   - 검색된 문맥(Context Pack)을 종합하여 시장 진입 가능성, 가격 포지셔닝, 차별화 USP 등이 포함된 **전략 보고서**를 생성합니다.
5. **Creative Agent (`creative_agent.py`)**
   - 확정된 전략을 바탕으로 SNS 플랫폼(인스타그램, 틱톡 등) 성격에 맞는 톡톡 튀는 **마케팅 카피**와 **DALL-E 3 제품 연출 포스터**를 생성합니다.

## ⚙️ 설치 및 세팅 방법

### 1. 패키지 설치
Python 3.10 이상 환경을 권장합니다.
```bash
pip install -r requirements.txt
```

### 2. 환경 변수 (.env) 설정
프로젝트 최상단(또는 `trendpilot` 폴더 내부)에 `.env` 파일을 만들고 아래 API 키들을 입력합니다.
```env
OPENAI_API_KEY=본인의_오픈AI_키를_입력
...
```

## 🚀 실행 순서

이 프로젝트는 먼저 로컬 전문 지식 데이터베이스(규제 및 수출 통계)를 구축한 뒤, 웹 인터페이스를 실행하는 2단계로 진행됩니다.

### STEP 1. 로컬 데이터베이스(Local KB) 구축하기
전문적인 마케팅 전략 수립을 위해서는 현지 규제(예: FDA 허용/금지 성분, 식약처 가이드 등) 및 수출 통계 자료가 필요합니다.
1. `trendpilot/data/local_sources/` 폴더 안에 관련된 **PDF, CSV, TXT 파일**을 자유롭게 넣어주세요.
2. 아래 명령어를 실행하여 로컬 벡터 데이터베이스(Chroma DB)를 인덱싱합니다.
```bash
python build_local_kb.py
```
> 실행이 완료되면 `trendpilot/vector_db/local_kb` 폴더에 DB가 안전하게 저장됩니다.

### STEP 2. 메인 에이전트 UI 실행하기
데이터베이스 구축이 완료되었다면 메인 웹 대시보드를 실행합니다.
```bash
python app.py
```
1. 터미널에 표시된 `http://127.0.0.1:7860` (또는 포트에 따라 다름) 주소로 접속합니다.
2. 화장품 제품의 사진을 업로드하고, 진출하고자 하는 **목표 국가**(예: 미국, 일본, 유럽 등)를 입력합니다.
3. **[🚀 패키지 자동 생성]** 버튼을 클릭합니다.
4. UI 우측 탭을 이동하며 1)제품 분석, 2)검색된 문맥, 3)전략 보고서, 4)크리에이티브 완성본이 순차적으로 도출되는 과정을 확인하세요!
